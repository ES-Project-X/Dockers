PACKAGE CONTENT
===============

This package contains OpenStreetMap data of the
land polygons, split into a grid with slight overlap.

Layers contained are:

land_polygons.shp:

  758284 Polygon features
  Mercator projection (EPSG: 3857)
  Extent: (-20037507, -20037507) - (20037508, 18460131)
  In geographic coordinates: (-180.000, -85.051) - (180.000, 83.665)

Date of the data used is 2023-10-11T00:00:00Z

You can find more information on this data set at

https://osmdata.openstreetmap.de/data/land-polygons.html
